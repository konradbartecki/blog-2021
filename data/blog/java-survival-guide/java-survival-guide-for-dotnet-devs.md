### LINQ = Stream()

| C# | Java   |
| ---- | ---- |
| `x => x.Method()` | `x -> x.Method()` |
| `Where` | `filter` |
| `Select` | `map` |
| `SelectMany` | `flatMap` |
| `ToList` | `collect(Collectors.toList())`
